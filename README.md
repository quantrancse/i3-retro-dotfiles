# Manjaro i3 retro
### Some details
+ **OS**: Manjaro
+ **WM**: i3
+ **Terminal**: termite
+ **Shell**: bash
+ **File Manager**: ranger, pcmanfm
+ **Launcher**: rofi
+ **Editors**: Spacevim, vscode
+ **Reader**: zathura (mupdf)
+ **Browser**: qutebrowser, Chrome
+ **Music**: mpc, mpd, ncmpcpp, mpv
+ **Theme**: Arc-Gruvbox

#### Others
+ polybar, fontawesome, fcitx,...
